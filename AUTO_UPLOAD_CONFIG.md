---

## 如何启用自动上传

### 方案1：提供私钥文件（推荐用于自动化部署）

如果您希望每次更新都自动上传到微信平台，需要：

#### 步骤1：获取上传私钥

1. 登录微信小程序后台：https://mp.weixin.qq.com/
2. 进入：开发 -> 开发管理 -> 开发设置
3. 找到"小程序代码上传"部分
4. 点击"生成"或"重置"私钥
5. 下载私钥文件（通常命名为 `private.key`）

#### 步骤2：上传私钥文件

将私钥文件上传到项目目录：

```bash
# 方式1：直接上传到 miniprogram/ 目录
/workspace/projects/miniprogram/private.key

# 方式2：上传到项目根目录
/workspace/projects/private.key
```

#### 步骤3：设置文件权限

```bash
chmod 600 /workspace/projects/miniprogram/private.key
```

#### 步骤4：告诉我

回复我私钥文件的位置，例如：
- "私钥文件在 /workspace/projects/miniprogram/private.key"
- "私钥文件在 /workspace/projects/private.key"

## 安全提醒

⚠️ **私钥文件安全注意事项**：

1. **不要将私钥文件提交到 Git 仓库**
   - 私钥文件应该放在 `.gitignore` 中
   - 确保私钥文件不会泄露

2. **设置正确的文件权限**
   - 私钥文件权限应该是 `600`（仅所有者可读写）
   - 避免其他用户访问

3. **定期更换私钥**
   - 建议定期在微信小程序后台重置私钥
   - 如果私钥泄露，立即在后台重置

4. **私钥使用范围**
   - 上传私钥只能用于代码上传
   - 不能用于其他操作（如获取用户信息等）

## 自动化工作流

如果配置了私钥，未来的更新流程将是：

```
1. 修复代码问题
   ↓
2. 打包新版本（tar.gz）
   ↓
3. 上传到对象存储（自动）
   ↓
4. 上传到微信平台（自动）
   ↓
5. 提供下载链接和版本信息
```

## 手动上传步骤

如果没有配置私钥，可以按照以下步骤手动上传：

### 详细步骤见：`UPLOAD_GUIDE.md`

1. 下载最新版本
2. 在微信开发者工具中打开
3. 测试小程序
4. 点击"上传"按钮
5. 填写版本信息和描述
6. 提交审核
