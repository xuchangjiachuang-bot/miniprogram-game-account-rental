## 方式二：使用命令行工具上传（需要私钥）

### 准备工作

1. **获取上传私钥**
   - 登录微信小程序后台：https://mp.weixin.qq.com/
   - 进入：开发 -> 开发管理 -> 开发设置
   - 找到"小程序代码上传"部分
   - 点击"生成"或"重置"私钥
   - 下载私钥文件（通常命名为 `private.key`）

2. **配置私钥文件**
   ```bash
   # 将私钥文件复制到项目目录
   cp /path/to/private.key /workspace/projects/miniprogram/
   ```

3. **设置文件权限**
   ```bash
   chmod 600 /workspace/projects/miniprogram/private.key
   ```

### 上传命令

```bash
cd /workspace/projects/miniprogram
/usr/bin/miniprogram-ci upload \
  --ppp ./private.key \
  --project . \
  --version 1.0.20260226.2137 \
  --desc "修复TabBar配置错误，解决小程序启动崩溃问题"
```

### 参数说明

- `--ppp`: 私钥文件路径
- `--project`: 项目目录
- `--version`: 版本号
- `--desc`: 版本描述

## 审核注意事项

### 审核前检查清单

- [ ] 所有页面可以正常访问
- [ ] 没有运行时错误
- [ ] 所有功能都可以正常使用（或使用Mock数据）
- [ ] 没有违反微信小程序审核规范的内容
- [ ] 隐私协议已设置
- [ ] 用户协议已设置
- [ ] 服务类目已配置

### 常见审核问题

1. **功能不完整**
   - 解决：确保所有页面和功能都可以正常访问

2. **数据为空**
   - 解决：使用Mock数据确保页面有内容显示

3. **缺少用户协议**
   - 解决：在小程序后台设置用户协议和隐私协议

4. **服务类目不匹配**
   - 解决：在小程序后台选择正确的服务类目
