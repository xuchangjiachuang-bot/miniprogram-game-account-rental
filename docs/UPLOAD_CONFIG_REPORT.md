## 🔄 上传流程

### 1. 检查配置（可选）

```bash
node check-upload-config.js
```

### 2. 选择上传方式

选择上述四种方式中的任意一种

### 3. 执行上传

```bash
# 示例：使用一键上传
./一键上传.sh
```

### 4. 查看上传结果

上传成功后会显示：
- ✅ 版本号
- ✅ 描述
- ✅ 上传时间
- ✅ AppID

### 5. 提交审核

1. 登录 https://mp.weixin.qq.com/
2. 进入：管理 → 版本管理
3. 找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息并提交

### 6. 发布

审核通过后：
1. 点击「发布」
2. 选择发布范围（全量或灰度）
3. 确认发布
