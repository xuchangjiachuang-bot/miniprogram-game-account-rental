✅ 已完成配置：

- miniprogram-ci 已安装
- 上传脚本已准备

## 🎯 四种上传方式

### 方式 1️⃣：一键上传（最简单）

```bash

```

**特点**：
- ✅ 最简单，一键完成
- ✅ 自动生成版本号
- ✅ 自动生成描述

**指定版本号和描述**：
```bash
./一键上传.sh 1.0.20260226.2200 "修复登录bug"
```

- ✅ 自动生成描述

**指定版本号和描述**：
```bash
cd miniprogram
```


**特点**：

- ✅ 自动检测私钥和配置
- ✅ 详细的错误提示

**指定版本号和描述**：

**指定版本号和描述**：
```bash

```

## 📊 版本号说明

### 自动生成规则

所有上传方式都支持自动生成版本号，格式为：

```
1.0.YYYYMMDD.HHMM
```

示例：
- `1.0.20260226.2200` - 2026年2月26日 22:00
- `1.0.20260301.0830` - 2026年3月1日 08:30

### 手动指定版本号

```bash
./一键上传.sh 1.2.3 "发布正式版"
```

## 🔄 上传后操作

### 1. 登录微信公众平台
访问：https://mp.weixin.qq.com/

### 2. 进入版本管理
路径：管理 → 版本管理

### 3. 查看上传的版本
找到刚上传的版本号

### 4. 提交审核
- 点击「提交审核」
- 填写审核信息
- 提交等待审核

### 5. 审核通过后发布
- 点击「发布」
- 选择发布范围
- 确认发布

## ⚠️ 常见问题

### Q1: 提示私钥文件不存在？

**A**: 确认私钥文件存在：
```bash
ls -la miniprogram/private.key
```

检查文件权限：
```bash
chmod 600 miniprogram/private.key
```

---

### Q2: 上传失败，提示 IP 白名单错误？

**A**: IP 白名单需要 5-10 分钟生效：
1. 确认 IP 已添加到微信后台
2. 等待 5-10 分钟
3. 重新尝试上传

---

### Q3: 上传进度卡住？

**A**: 可能原因：
1. 网络连接问题
2. 微信服务器繁忙
3. 项目文件过大

解决方案：
- 检查网络连接
- 等待几分钟后重试
- 检查 `ignores` 配置是否正确

---

### Q4: 版本号已存在？

**A**: 使用不同的版本号：
```bash
./一键上传.sh 1.0.20260226.2201 "新版本"
```

或手动指定：
```bash
./一键上传.sh 1.0.1 "手动版本号"
```

---

### Q5: 上传成功但找不到版本？

**A**: 检查：
1. 是否登录正确的微信小程序账号
2. 是否在微信公众平台查看
3. AppID 是否配置正确
4. 刷新页面
