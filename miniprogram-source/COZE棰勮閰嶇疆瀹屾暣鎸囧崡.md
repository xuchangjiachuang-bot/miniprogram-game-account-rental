# 📱 Coze 小程序预览配置指南

根据 Coze 官方文档，以下是配置小程序预览的详细步骤。

## 🎯 关于 Coze 小程序预览协议

Coze 提供了小程序预览功能，允许开发者通过绑定 AppID 的方式生成预览二维码，用于真机调试。

**重要说明**：
- Coze 编程主要是通过 AI 对话自动生成小程序代码
- 您当前已手动开发完成小程序代码（在 `miniprogram/` 目录）
- 有两种使用方式：

1. **方式一**：使用 Coze AI 重新开发小程序（推荐新项目）
2. **方式二**：使用微信开发者工具预览当前项目（推荐现有项目）

---

## 🚀 方式一：通过 Coze 编程预览（AI 自动开发）

### 步骤 1：访问 Coze 编程

1. 访问 Coze 官网：https://www.coze.cn
2. 登录您的账号
3. 在首页点击"小程序"选项卡
4. 创建新的小程序项目

### 步骤 2：输入需求描述

在文本框中输入详细的小程序需求描述：

```
帮我开发一个游戏账号租赁平台小程序，主要功能包括：

1. 用户认证：
   - 微信登录
   - 手机号绑定

2. 首页：
   - 轮播图展示
   - 账号列表（支持筛选：平台、哈夫币、段位、安全箱、皮肤）
   - 卡片式布局展示账号信息

3. 订单模块：
   - 订单列表（待付款、租赁中、已完成等状态）
   - 订单操作（支付、取消、聊天、续租）

4. 账号模块：
   - 账号详情页（图片轮播、属性展示、收藏、分享）
   - 账号发布

5. 钱包模块：
   - 余额展示
   - 充值提现
   - 交易记录

6. 聊天模块：
   - 聊天列表
   - 实时消息

7. 个人中心：
   - 用户信息
   - 订单管理
   - 账号管理
   - 设置

设计要求：
- 使用紫色主题色（#667eea）
- 现代化卡片式设计
- 流畅的交互动画

后端要求：
- 调用 Next.js API Routes
- 使用 WebSocket 实现实时聊天
- PostgreSQL 数据库
```

### 步骤 3：绑定微信小程序

Coze AI 生成代码后：

1. 在页面右下角点击 **"AppID 设置"**
2. 填写 AppID：`twx2382e1949d031ba6`
3. 点击 **"绑定"**
4. 使用小程序管理员微信号扫描授权二维码
5. 在微信中完成授权（选择小程序 → 确认授权）

### 步骤 4：生成预览二维码

授权成功后：

1. 在页面右侧找到预览二维码
2. 使用管理员微信号扫描二维码
3. 在微信中预览小程序

### 步骤 5：真机调试

在手机上测试小程序：

- ✅ 功能是否可用
- ✅ 交互是否完善
- ✅ 样式是否符合预期
- ✅ AI 能力是否正常（如适用）

## 🛠️ 方式二：通过微信开发者工具预览（推荐）

由于您已经手动开发完成小程序代码，推荐使用微信开发者工具：

### 步骤 1：安装微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 步骤 2：导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 项目目录：`/workspace/projects/miniprogram`
4. AppID：`twx2382e1949d031ba6`
5. 点击"导入"

### 步骤 3：配置服务器域名

登录微信公众平台（https://mp.weixin.qq.com）：

1. 进入：开发 → 开发管理 → 开发设置 → 服务器域名
2. 配置以下域名：

```
request 合法域名：
- https://your-api-domain.com  (替换为实际后端API域名)

socket 合法域名：
- wss://your-socket-domain.com  (替换为实际WebSocket域名)

uploadFile 合法域名：
- https://your-api-domain.com

downloadFile 合法域名：
- https://your-api-domain.com
```

### 步骤 4：配置后端地址

编辑 `miniprogram/utils/config.js`：

```javascript
module.exports = {
  // 替换为实际的后端API地址
  API_BASE_URL: 'https://your-api-domain.com',
  
  // 替换为实际的WebSocket地址
  WEBSOCKET_URL: 'wss://your-socket-domain.com',
  
  // AppID
  APPID: 'twx2382e1949d031ba6',
  
  // 其他配置...
};
```

### 步骤 5：生成预览二维码

1. 点击工具栏的"预览"按钮
2. 使用微信扫描生成的二维码
3. 在手机上预览小程序
