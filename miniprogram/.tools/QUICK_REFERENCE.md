## 🚀 两个选择

### 选择1：配置完整的生产环境（推荐）⭐

**适用情况**：您有服务器权限，希望小程序正式上线

**需要做的事**：
1. ✅ 配置域名解析（A记录指向服务器IP）
2. ✅ 安装配置Nginx（反向代理到localhost:5000）
3. ✅ 获取SSL证书（HTTPS）
4. ✅ 配置微信小程序白名单
5. ✅ 重新上传小程序

**耗时**：约40分钟
**效果**：✅ 完全可用

**详细文档**：`PRODUCTION_SETUP_GUIDE.md`

---

### 选择2：使用Mock数据（临时方案）⚡

**适用情况**：暂时无法配置域名，只想测试界面

**需要做的事**：
1. ✅ 修改配置文件开启Mock数据（1行代码）
2. ✅ 重新上传小程序

**耗时**：约3分钟
**效果**：⚠️ 仅界面可用，无法进行真实交易

**详细说明**：

修改文件：`miniprogram/utils/config.js`

```javascript
// 找到这一行（大约第20行）
production: {
  baseUrl: 'https://yugioh.top/api',
  wsUrl: 'wss://yugioh.top',
  debug: false,
  useMockData: false,  // 🔴 改为 true
},
```

然后重新上传：
```bash
cd /workspace/projects/miniprogram
node upload-simple.js
```

## 💡 关键提示

### 关于生产环境

- 小程序上线必须使用HTTPS域名
- 域名必须备案（中国大陆服务器）
- 不支持IP地址

### 关于Mock数据

- 仅用于测试界面
- 数据是假的，不要用于真实交易
- 后续可以切换到真实API

## 🚨 常见问题

**Q1: 我不知道服务器的公网IP怎么办？**

A: 执行以下命令查看：
```bash
curl ifconfig.me
# 或者
curl ip.sb
```

**Q2: 我没有域名怎么办？**

A: 可以购买域名，或者暂时使用Mock数据。

**Q3: 域名备案需要多久？**

A: 通常1-3周，具体取决于管局。

**Q4: Mock数据可以一直用吗？**

A: 不建议。Mock数据只是临时方案，正式上线需要配置真实API。
