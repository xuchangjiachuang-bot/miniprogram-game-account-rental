## 技术要点

### 1. Drizzle ORM 使用
- 使用 `pgTable` 定义表结构
- 使用 `db.insert()`, `db.select()`, `db.update()` 进行数据库操作
- 使用 `eq()`, `and()`, `desc()` 等函数构建查询条件

### 2. 微信支付流程
1. 前端调用 `createMinipPayment` 创建支付订单
2. 后端调用微信统一下单接口
3. 返回 JSAPI 支付参数
4. 前端调用 `wx.requestPayment` 发起支付
5. 微信回调通知后端支付结果
6. 后端更新订单状态和账号状态

### 3. 小程序支付注意事项
- `package` 是 JavaScript 保留字，不能用作变量名
- 需要使用 `package: pkg` 重命名
- 需要在微信开放平台配置支付目录和域名

### 4. Token 认证
- 使用 `Authorization: Bearer <token>` 传递用户认证信息
- 临时方案直接使用 token 作为 userId（生产环境需要验证）
