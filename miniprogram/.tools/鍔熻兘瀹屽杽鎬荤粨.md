## 数据库表结构

### 核心表
- `userBalances` - 用户余额表
- `balanceTransactions` - 余额变动记录表
- `withdrawals` - 提现申请表
- `verificationApplications` - 实名认证表

## API 端点

### 钱包相关
- `GET /api/wallet/statistics` - 获取钱包统计数据
- `GET /api/wallet/bills` - 获取月度账单
- `GET /api/wallet` - 获取钱包信息
- `GET /api/wallet/transactions` - 获取交易记录

### 提现相关
- `POST /api/withdrawals` - 创建提现申请
- `GET /api/withdrawals` - 获取提现记录
- `GET /api/withdrawals/:id` - 获取提现详情

### 实名认证相关
- `POST /api/verification/initiate` - 提交实名认证申请
- `GET /api/verification/result` - 查询实名认证结果
- `GET /api/verification/history` - 获取实名认证历史记录
