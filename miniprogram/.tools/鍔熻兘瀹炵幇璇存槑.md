# 微信小程序功能实现说明

## 已实现功能

### 1. 微信授权登录 ✅

**文件位置**: `miniprogram/pages/auth/wechat-login/`

**功能描述**:
- 使用微信 `getUserInfo` 接口获取用户授权
- 调用后端 `wechatLogin` 接口完成登录
- 保存 token 和用户信息到本地存储
- 登录成功后跳转到首页

**使用方式**:
```javascript
// 在需要登录的地方调用
wx.navigateTo({
  url: '/pages/auth/wechat-login/index'
});
```

**后端接口**:
```
POST /api/auth/wechat/login
参数:
  - code: 微信登录code
  - userInfo: 用户信息
  - encryptedData: 加密数据
  - iv: 初始化向量
返回:
  - token: 登录凭证
  - userInfo: 用户信息
```

### 2. 获取手机号 ✅

**文件位置**: `miniprogram/pages/auth/bind-phone/`

**功能描述**:
- 使用微信 `getPhoneNumber` 接口获取用户手机号授权
- 调用后端 `bindPhone` 接口绑定手机号
- 保存手机号到用户信息
- 绑定成功后跳转到首页

**使用方式**:
```javascript
// 需要绑定手机号时调用
wx.navigateTo({
  url: '/pages/auth/bind-phone/index'
});
```

**后端接口**:
```
POST /api/auth/bind-phone
参数:
  - code: 微信手机号code
  - encryptedData: 加密数据
  - iv: 初始化向量
返回:
  - phone: 手机号
  - success: 是否成功
```

**实现代码示例**:
```javascript
// pages/auth/bind-phone/index.js
Page({
  onGetPhoneNumber(e) {
    if (e.detail.errMsg !== 'getPhoneNumber:ok') {
      return;
    }
    
    const { code, encryptedData, iv } = e.detail;
    
    api.bindPhone({ code, encryptedData, iv })
      .then(res => {
        wx.showToast({
          title: '绑定成功',
          icon: 'success'
        });
        
        // 更新用户信息
        storage.getUserInfo().then(userInfo => {
          if (userInfo) {
            userInfo.phone = res.phone;
            storage.setUserInfo(userInfo);
          }
        });
        
        wx.switchTab({
          url: '/pages/index/index'
        });
      })
      .catch(error => {
        wx.showToast({
          title: error.error || '绑定失败',
          icon: 'none'
        });
      });
  }
});
```

### 3. 微信支付（待实现）

**需要创建的文件**:
- `miniprogram/pages/order/payment/wechat/index.js`
- `miniprogram/pages/order/payment/wechat/index.wxml`
- `miniprogram/pages/order/payment/wechat/index.wxss`

**功能描述**:
- 调用 `wx.requestPayment` 发起支付
- 支付成功后更新订单状态
- 支付失败提示用户

**后端接口**:
```
POST /api/payment/wechat/create
参数:
  - orderId: 订单ID
返回:
  - timeStamp: 时间戳
  - nonceStr: 随机字符串
  - package: 订单详情
  - signType: 签名方式
  - paySign: 签名

POST /api/payment/wechat/notify
参数:
  - 微信支付回调参数
返回:
  - success: 是否成功
```

**实现代码示例**:
```javascript
// pages/order/payment/wechat/index.js
Page({
  onLoad(options) {
    const { orderId } = options;
    this.createPayment(orderId);
  },

  createPayment(orderId) {
    const that = this;
    
    api.createWechatPayment({ orderId })
      .then(res => {
        const { timeStamp, nonceStr, package, signType, paySign } = res.data;
        
        wx.requestPayment({
          timeStamp,
          nonceStr,
          package,
          signType,
          paySign,
          success(res) {
            console.log('支付成功:', res);
            wx.navigateTo({
              url: `/pages/order/success/index?id=${orderId}`
            });
          },
          fail(err) {
            console.error('支付失败:', err);
            wx.showToast({
              title: '支付失败',
              icon: 'none'
            });
          }
        });
      })
      .catch(error => {
        wx.showToast({
          title: error.error || '创建订单失败',
          icon: 'none'
        });
      });
  }
});
```

### 5. 提现功能（待实现）

**需要创建的文件**:
- `miniprogram/pages/wallet/withdraw/index.js`
- `miniprogram/pages/wallet/withdraw/index.wxml`
- `miniprogram/pages/wallet/withdraw/index.wxss`

**功能描述**:
- 输入提现金额
- 选择提现方式（微信）
- 输入提现账号
- 提交提现申请
- 显示提现记录

**后端接口**:
```
POST /api/wallet/withdraw/create
参数:
  - amount: 提现金额
  - method: 提现方式（wechat）
  - account: 提现账号
返回:
  - withdrawId: 提现ID

GET /api/wallet/withdraw/list
参数:
  - page: 页码
  - pageSize: 每页数量
返回:
  - list: 提现记录列表
```

**实现代码示例**:
```javascript
// pages/wallet/withdraw/index.js
Page({
  data: {
    amount: '',
    availableBalance: 0,
    loading: false
  },

  onLoad() {
    this.loadWalletInfo();
  },

  loadWalletInfo() {
    api.getWalletInfo().then(res => {
      this.setData({
        availableBalance: res.data.availableBalance
      });
    });
  },

  onWithdraw() {
    const { amount, availableBalance } = this.data;
    
    if (!amount || parseFloat(amount) <= 0) {
      wx.showToast({
        title: '请输入提现金额',
        icon: 'none'
      });
      return;
    }
    
    if (parseFloat(amount) > availableBalance) {
      wx.showToast({
        title: '余额不足',
        icon: 'none'
      });
      return;
    }
    
    this.setData({ loading: true });
    
    api.createWithdraw({
      amount: parseFloat(amount),
      method: 'wechat'
    })
    .then(res => {
      wx.showToast({
        title: '提现申请已提交',
        icon: 'success'
      });
      
      setTimeout(() => {
        wx.navigateBack();
      }, 1500);
    })
    .catch(error => {
      wx.showToast({
        title: error.error || '提现失败',
        icon: 'none'
      });
      this.setData({ loading: false });
    });
  }
});
```

### 7. 实名认证（待实现）

**需要创建的文件**:
- `miniprogram/pages/profile/verification/index.js`
- `miniprogram/pages/profile/verification/index.wxml`
- `miniprogram/pages/profile/verification/index.wxss`

**功能描述**:
- 输入真实姓名
- 输入身份证号
- 上传身份证照片（正反面）
- 提交认证申请
- 等待审核

**后端接口**:
```
POST /api/user/verification/submit
参数:
  - realName: 真实姓名
  - idCard: 身份证号
  - idCardFront: 身份证正面照片
  - idCardBack: 身份证反面照片
返回:
  - verificationId: 认证ID

GET /api/user/verification/status
返回:
  - status: 认证状态（pending/approved/rejected）
  - reason: 拒绝原因
```

**实现代码示例**:
```javascript
// pages/profile/verification/index.js
Page({
  data: {
    realName: '',
    idCard: '',
    idCardFront: '',
    idCardBack: '',
    loading: false
  },

  onChooseIdCardFront() {
    wx.chooseImage({
      count: 1,
      success(res) {
        this.setData({ idCardFront: res.tempFilePaths[0] });
      }
    });
  },

  onChooseIdCardBack() {
    wx.chooseImage({
      count: 1,
      success(res) {
        this.setData({ idCardBack: res.tempFilePaths[0] });
      }
    });
  },

  onSubmit() {
    const { realName, idCard, idCardFront, idCardBack } = this.data;
    
    if (!realName || !idCard || !idCardFront || !idCardBack) {
      wx.showToast({
        title: '请填写完整信息',
        icon: 'none'
      });
      return;
    }
    
    this.setData({ loading: true });
    
    // 上传图片
    Promise.all([
      this.uploadImage(idCardFront),
      this.uploadImage(idCardBack)
    ])
    .then(([frontUrl, backUrl]) => {
      return api.submitVerification({
        realName,
        idCard,
        idCardFront: frontUrl,
        idCardBack: backUrl
      });
    })
    .then(res => {
      wx.showToast({
        title: '认证已提交，等待审核',
        icon: 'success'
      });
      
      setTimeout(() => {
        wx.navigateBack();
      }, 1500);
    })
    .catch(error => {
      wx.showToast({
        title: error.error || '提交失败',
        icon: 'none'
      });
      this.setData({ loading: false });
    });
  },

  uploadImage(filePath) {
    return new Promise((resolve, reject) => {
      wx.uploadFile({
        url: api.getBaseUrl() + '/api/upload/image',
        filePath,
        name: 'file',
        header: {
          'Authorization': 'Bearer ' + storage.getToken()
        },
        success(res) {
          const data = JSON.parse(res.data);
          if (data.success) {
            resolve(data.data.url);
          } else {
            reject(new Error(data.error));
          }
        },
        fail: reject
      });
    });
  }
});
```

## 注意事项

1. **微信支付需要配置**:
   - 在微信小程序后台配置商户号
   - 配置支付密钥
   - 配置支付回调地址

2. **企业微信客服需要配置**:
   - 在企业微信后台配置客服
   - 获取 corpId 和 kfId
   - 配置客服接入地址

3. **实名认证需要审核**:
   - 用户提交后需要人工或自动审核
   - 审核结果需要及时通知用户

4. **安全性**:
   - 所有敏感信息必须通过 HTTPS 传输
   - Token 需要定期刷新
   - 支付和提现需要二次验证
