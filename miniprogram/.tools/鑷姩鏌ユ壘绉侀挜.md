# ✅ 私钥文件自动查找机制

## 🎯 自动查找原理

所有上传脚本都已配置为**自动查找私钥文件**，无需每次手动指定路径。

### 查找逻辑

所有脚本使用统一的查找优先级：

```
1. 优先使用环境变量 MINIPROGRAM_PRIVATE_KEY_PATH（如果设置了）
2. 否则自动查找项目默认路径：miniprogram/private.key
```

### 脚本配置详情

#### 1. upload-miniprogram.sh
```bash
PRIVATE_KEY="${MINIPROGRAM_PRIVATE_KEY_PATH:-$SCRIPT_DIR/miniprogram/private.key}"
```

#### 2. upload-wechat-miniprogram.js
```javascript
privateKeyPath: process.env.MINIPROGRAM_PRIVATE_KEY_PATH || path.join(__dirname, 'miniprogram', 'private.key')
```

#### 3. 一键上传.sh
调用 `upload-miniprogram.sh`，继承其自动查找逻辑

## 📁 私钥文件位置

**永久保存位置**：
```
/workspace/projects/miniprogram/private.key
```

**文件权限**：`600`（仅所有者可读写）

## 🚀 使用方式

### 无需配置，直接使用

```bash
# 方式1：一键上传
./一键上传.sh

# 方式2：npm 脚本
cd miniprogram && npm run upload

# 方式3：Shell 脚本
bash upload-miniprogram.sh

# 方式4：Node.js 脚本
node upload-wechat-miniprogram.js
```

**所有上述命令都会自动找到私钥文件！**

### 指定版本号（可选）

```bash
# 自动生成版本号
./一键上传.sh

# 指定版本号
./一键上传.sh 1.0.20260226.2210

# 指定版本号和描述
./一键上传.sh 1.0.20260226.2210 "修复xxx问题"
```

## ⚙️ 环境变量（可选）

如果需要自定义私钥路径，可以设置环境变量：

```bash
export MINIPROGRAM_PRIVATE_KEY_PATH=/custom/path/private.key
./一键上传.sh
```

**注意**：通常不需要设置，默认路径已经配置好。
