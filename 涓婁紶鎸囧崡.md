# 🚀 微信小程序快速上传

## ✅ 私钥已永久配置

私钥文件已保存在项目中，所有上传脚本会**自动查找**，无需手动指定。

## ⚡ 一键上传

```bash
# 进入项目目录
cd /workspace/projects

# 一键上传（自动生成版本号）
./一键上传.sh

# 指定版本号和描述
./一键上传.sh 1.0.20260226.2210 "本次更新内容"
```

## 📦 其他方式

### npm 脚本
```bash
cd miniprogram
npm run upload
```

### Shell 脚本
```bash
bash upload-miniprogram.sh
```

### Node.js 脚本
```bash
node upload-wechat-miniprogram.js
```

## 📋 上传后操作

1. 登录微信小程序后台：https://mp.weixin.qq.com/
2. 进入：管理 → 版本管理
3. 查看最新上传的版本
4. 点击"提交审核"
5. 填写审核信息并提交
