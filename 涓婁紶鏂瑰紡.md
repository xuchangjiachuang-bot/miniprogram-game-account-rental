# 🚀 小程序快速上传方式

私钥已永久配置，可以使用以下任意方式快速上传：

## 1️⃣ 最简单：一键上传

```bash
./一键上传.sh
```

## 2️⃣ 推荐方式：npm 脚本

```bash
cd miniprogram
npm run upload
```

## 3️⃣ Shell 脚本

```bash
bash upload-miniprogram.sh
```

## 4️⃣ Node.js 脚本

```bash
node upload-wechat-miniprogram.js
```
