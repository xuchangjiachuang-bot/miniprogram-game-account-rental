# 账号管理功能说明

## 访问路径

### 方式一：通过导航栏（推荐）
1. 登录账号
2. 点击右上角用户头像
3. 在下拉菜单中点击 **"我的账号"**

### 方式二：直接访问
访问URL：`/seller/accounts`

### 方式三：通过账号发布按钮
1. 点击导航栏的 **"账号发布"** 按钮
2. 在页面左上角有 **"我的账号"** 按钮

## 功能说明

### 1. 账号列表
页面会显示所有已上架的账号，包括：
- 账号名称
- 哈夫币数量
- 段位
- 租金、押金、总价
- 账号状态
- 账号标签

### 2. 账号状态

| 状态 | 说明 | 是否可编辑 |
|------|------|-----------|
| 待审核 | 新上架的账号，等待平台审核 | ❌ 不可编辑 |
| 在售中 | 审核通过，正在出售中 | ✅ 可编辑（仅限未售出） |
| 已售出 | 已被买家购买 | ❌ 不可编辑 |
| 已拒绝 | 审核未通过 | ❌ 不可编辑 |

### 3. 编辑账号

#### 可编辑条件
- 账号状态为 **"在售中"**
- 且 **销售次数为 0**（即从未被购买过）

#### 编辑步骤
1. 在账号列表中找到要编辑的账号
2. 点击 **"编辑"** 按钮
3. 在弹出的对话框中修改信息
4. 点击 **"保存修改"** 按钮

#### 可编辑内容
- 账号名称
- 哈夫币数量
- 账号等级
- 段位
- KD值
- 安全箱
- 体力等级
- 负重等级
- 登录方式
- 省份、城市
- 可租时间段
- 账号描述
- 标签

### 4. 删除账号

#### 删除条件
- 账号状态为 **"在售中"**
- 且 **销售次数为 0**

#### 删除步骤
1. 在账号列表中找到要删除的账号
2. 点击红色的 **删除按钮**（垃圾桶图标）
3. 确认删除

### 5. 筛选和搜索

#### 状态筛选
- 全部状态
- 待审核
- 在售中
- 已拒绝
- 已售出

#### 搜索
可以搜索：
- 账号名称
- 账号ID

### 6. 统计信息
页面顶部显示统计卡片：
- 总账号数
- 待审核数量
- 在售中数量
- 已售出数量

### 7. 上架新账号
点击右上角的 **"上架新账号"** 按钮，跳转到账号上架页面。

## 注意事项

### 编辑限制
⚠️ **重要提醒**：
1. 只有 **在售中** 且 **从未被购买** 的账号才能编辑
2. 账号一旦被购买，就 **无法编辑或删除**
3. 审核中或已拒绝的账号需要联系客服处理

### 销售次数
- 销售次数为 0：账号从未被租赁过，可以编辑/删除
- 销售次数 > 0：账号已被租赁过，无法编辑/删除

### 账号状态流转
```
待审核 → 审核通过 → 在售中 → 被购买 → 已售出
         ↓
       审核拒绝 → 已拒绝（需联系客服）
```

## 常见问题

### Q1: 为什么我的账号不能编辑？
A: 请检查账号状态和销售次数：
- 状态必须是"在售中"
- 销售次数必须为 0

### Q2: 账号被购买后还能修改吗？
A: 不能。账号一旦被购买，就锁定信息，保证买家租赁期间信息稳定。

### Q3: 审核未通过的账号怎么办？
A: 需要联系客服，说明情况，重新提交审核。

### Q4: 如何上架新账号？
A: 点击页面右上角的"上架新账号"按钮，填写账号信息并提交。

### Q5: 如何查看账号的销售情况？
A: 账号卡片上会显示销售次数，也可以通过"我的订单"查看订单详情。
